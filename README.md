# School CRUD Application

A Java Servlet based CRUD application using JDBC and Oracle Database.

## Technologies Used
- Java
- Servlets
- JDBC
- Oracle Database
- HTML & CSS
- GlassFish

## Features
- Add School
- Update School
- Delete School
- View School by ID

## How to Run
1. Configure Oracle DB
2. Deploy project on Apache Tomcat
3. Access via browser
